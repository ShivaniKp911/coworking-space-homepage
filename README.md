# Coworking Space Homepage

This project is based on a Scrimba tutorial. After completing the tutorial, I rebuilt the page from scratch to practice the concepts on my own.

## Purpose

The goal of this project was to strengthen my understanding of CSS positioning while recreating a simple coworking space homepage.

## Tech Stack

- HTML
- CSS

## Concepts Practiced

- `position: relative`
- `position: absolute`
- `position: fixed`

## Key Takeaways

- `absolute` elements position themselves relative to the nearest positioned ancestor.
- `relative` positioning is often used to create a positioning context for child elements.
- `fixed` positioning keeps elements attached to the viewport even while scrolling.
